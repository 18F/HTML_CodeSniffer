Success Criterion 2.4.10: Section Headings (Level AAA)

Section headings are used to organize the content.

Note 1: "Heading" is used in its general sense and includes titles and other ways to add a heading to different types of content.

Note 2: This success criterion covers sections within writing, not user interface components. User Interface components are covered under Success Criterion 4.1.2.

(From Understanding WCAG 2.0: Understanding Success Criterion 2.4.10)

Note

No specific test file provided. Where HTML is used, testing for relevant semantic headers (h1-h6) occurs in Success Criterion 1.3.1.
