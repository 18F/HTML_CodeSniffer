WCAG 2.0 Success Criterion 2.4.9: Link Purpose (Link Only) (Level AAA)

Back to: Home - Level AAA Summary

Definition

A mechanism is available to allow the purpose of each link to be identified from link text alone, except where the purpose of the link would be ambiguous to users in general.

More discussion from W3C: Understanding Success Criterion 2.4.9

Summary

This is a stricter version of Success Criterion 2.4.4. At AAA level, the context around a link - such as the remaining text in a sentence, cell or list item - cannot be used for claiming that link text describes the purpose of the link. Also see that Success Criterion for a discussion on the phrase "ambiguous to users in general".
