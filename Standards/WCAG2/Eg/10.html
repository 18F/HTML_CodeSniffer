HTML_CodeSniffer Test Example 10

An img element must follow the guidelines of Success Criterion 1.1.1: Non-Text Content with regards usage of alt and title attributes.

This example uses empty alt text, but also provides a title attribute. Non-breaking spaces are not counted as spaces for the purposes of whether an attribute is empty. As per Example 6, empty alt attribute and non-empty title attribute is incorrect.

Code

<img src="htmlcs.png" alt="" title="&nbsp;" />
