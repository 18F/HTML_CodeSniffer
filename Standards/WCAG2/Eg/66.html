HTML_CodeSniffer Test Example 66

Success Criterion 3.1.2 (Language of Parts) states that the changes in language should be marked using language attributes to qualify for AA conformance.

This example should test the usage of non-well-formed lang and xml:lang attributes on a normal element. Two errors should appear: one for the lang element and one for the xml:lang.

Code

<div lang="en-AU-Latn" xml:lang="en-AU-Latn">Script and language in wrong order. Should be "en-Latn-AU"</div>
<div lang="x" xml:lang="x">"x" on its own. Private use section needs data after it.</div>
<div lang="i" xml:lang="i">"i" on its own. Irregular tag identifer needs data after it.</div>
