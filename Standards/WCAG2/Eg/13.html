According to Success Criterion 1.3.1: Info and Relationships, radio buttons or check boxes that are part of the same group - that is, using the same name attribute - should be using fieldset elements to group the options as a unit. This allows users to skip the entire group.

Code

<form>
    <input type="radio" id="radio1_1" name="radio1" value="1" /> <label for="radio1_1">Option 1</label>
    <input type="radio" id="radio1_2" name="radio1" value="2" /> <label for="radio1_2">Option 2</label>
</form>
